# GPT-5 released

On **7 August 2025**, [[OpenAI]] launched [[GPT-5]] across ChatGPT, Microsoft
Copilot, and the OpenAI API.[^openai-gpt5][^wiki-gpt5] Rather than exposing separate
model pickers, GPT-5 is a unified system — a fast, high-throughput model, a deeper
[[Reasoning models|reasoning]] model, and a real-time **router** that decides which
to use per query based on the conversation's complexity, tool needs, and explicit
user intent.[^wiki-gpt5] It shipped to developers in three API sizes — `gpt-5`,
`gpt-5-mini`, and `gpt-5-nano` — trading off capability, cost, and latency.[^openai-gpt5-dev]
