# Kimi K3 released

On **16 July 2026**, [[Moonshot AI]] released [[Kimi|Kimi K3]] — a **2.8-trillion-parameter**
sparse [[Mixture of Experts|MoE]] model that Moonshot billed as the world's first open "3T-class"
system, widely reported as the largest open-weight model shipped to date, available immediately
via web app and API.[^kimi-k3-blog][^wiki-kimi][^tomshardware-k3] It debuted **3rd overall** on
the Artificial Analysis leaderboard — behind [[Claude Fable 5]] and [[GPT-5|GPT-5.6 Sol]] — but
topped LMArena's Frontend Code Arena.[^wiki-kimi][^kimi-k3-blog][^willison-k3] [[Open weights]]
were promised by **27 July 2026**, not yet public as of this writing.[^kimi-k3-blog][^willison-k3]
