# OpenAI Codex launched

On **16 May 2025**, [[OpenAI]] launched **[[Codex]]**, a cloud-based agentic coding
tool powered by **codex-1** — a version of its o3 [[Reasoning models|reasoning
model]] tuned for software engineering — following the open-source Codex CLI released
the month before.[^codex_intro2025][^codex_wiki] It marked OpenAI's headline entry
into the autonomous coding-agent wave, opposite Anthropic's [[Claude Code]].
