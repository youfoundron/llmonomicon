# Cursor

**Cursor** is a proprietary, AI-first code editor built by the startup
**Anysphere** — a fork of Visual Studio Code that wraps frontier language models
into the editor itself. More than any other product, it brought AI-assisted
coding into the mainstream of professional development, pairing fast inline
autocomplete with an [[Agent|agent]] mode that can edit across an entire codebase
from natural-language instructions.[^cursor-wiki][^cursor-site]

## What it is

Cursor lets developers "edit code, search codebases, run commands, and complete
programming tasks using natural-language instructions."[^cursor-wiki] Beyond
ordinary autocomplete, it offers a chat interface and an **agent mode** that plans
and applies changes across multiple files.[^cursor-wiki] Anysphere — founded in
2022 by four MIT students, with Cursor launching publicly around 2023 — now
markets it simply as "your coding agent for building ambitious software."[^cursor-wiki][^cursor-site]

## Wrapping frontier models

A defining choice is that Cursor does not primarily ship its own models — it
**wraps existing ones**. It "integrates third-party large language models,
including models from Anthropic and OpenAI, in addition to its own models for
coding tasks,"[^cursor-wiki] letting users "choose between every cutting-edge
model from OpenAI, Anthropic, Gemini, xAI, and Cursor."[^cursor-site] That makes
it a model-agnostic IDE — in contrast to [[Claude Code]], which is
[[Anthropic]]'s own terminal-native agent, and to the open-source [[Aider]].
Among [[Agent harnesses]], Cursor is the breakout *closed, IDE-based* coding
agent.

## Scale and the SpaceX deal

Cursor grew exceptionally fast, reaching billions of dollars in annualized revenue
within a few years. In June 2026, SpaceX **agreed to acquire** Anysphere in an
all-stock deal valuing it at roughly $60 billion — announced on June 16, 2026 and,
at the time of writing, **not yet closed**: it is expected to complete in the
third quarter, pending regulatory approval.[^cnbc-spacex][^tc-spacex] At signing,
Cursor reported around $2.6 billion in annualized revenue.[^cnbc-spacex]
